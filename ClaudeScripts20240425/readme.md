## Contents

| File | Purpose |
|---|---|

## What the compilation indexes

Sixteen peer-citable preprints published on Zenodo under the Redacted Science Research Initiative, plus the Paper Seed Master and Architect Reports generated through the Seed Analysis Process. Each paper provides a *Summary for the Reader from the Architect* (plain-English entry point with bracketed asides) and the published *Abstract*. All papers are citable via DOI.

The framework treats *Candida albicans* as a coevolved fungal biochemical computer integrated into human physiology. Architect Reports extend the framework into specific subjects via a reproducible AI-assisted methodology documented in the Seed Master.

## Build script architecture

The script is data-driven. Adding or modifying a paper means editing the `papers[]` array, not touching rendering logic.

Each paper object accepts:

```javascript
{
  number,       // sequential numeric position (or omit if using prefix)
  prefix,       // e.g. 'AS#001' for Architect Reports
  title,        // paper title
  doi,          // Zenodo DOI without the https://doi.org/ prefix
  refs,         // reference count (rendered as "— N references")
  protocolDoi,  // optional, renders an indented sub-row (Clear Evidence pattern)
  summary,      // array of paragraph strings, supports [*italic*] and [bracket] markup
  abstract,     // array of paragraph strings; pass [] to suppress the section
  seedFooter,   // optional, renders sage-green footer (Architect Reports)
  isSeedMaster, // flag for the methodology document entry
  isArchitectReport, // flag for sage-green styling
  seriesGroup   // e.g. 'AS# — Stuck State Series' (renders as sub-heading)
}
```

### Critical: `doiLine(doi, refs)`

Reference count must be passed as a **separate parameter**. Concatenating it into the DOI string breaks the hyperlink (the entire string becomes the URL).

```javascript
// Correct
doiLine('10.5281/zenodo.19337525', 52)

// Broken — produces invalid hyperlink
doiLine('10.5281/zenodo.19337525 — 52 references')
```

## HTML compilation

The HTML is single-file and self-contained. No build step required — edit, deploy. Includes embedded CSS, JSON-LD ItemList schema, sitemap-aware canonical URL, and a revision history dropdown.

Sections:
1. **Foundational + Framework Papers** (Papers 1–13)
2. **Clinical Case Reports** (Papers 14–16, soft-grouped)
3. **The Paper Seed Master + Architect Reports** (sage-green accent, Nostr Event ID + njump.me view links per report)

## Architect Report numbering

Sequential across all series, prefix encodes type:

- `AM#` — Mechanism
- `AS#` — Stuck State
- `AB#` — Substance
- `AI#` — Autoimmune Collateral
- `AH#` — Historical / Institutional
- `AO#` — Observational

Gaps in sequence are intentional and reflect seed insertion order in the Seed Master.
